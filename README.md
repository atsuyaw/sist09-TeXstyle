# sist09
## これは何？
sist09（科学技術情報流通技術基準 科学技術レポートの様式）に準拠したレイアウトを行うためのスタイルファイルです。

## コマンドについて
いくつかのコマンドを導入しています。  
コマンドリストは[説明文書（`./sist09.pdf`）](./sist09.pdf)にありますのでそちらをご覧ください。

ドキュメントシートを出力する機能がありますが，現状では充分に実装していません。  
将来的に対応するつもりです。
